# 🚚 CargoOpt — Knapsack-Based Cargo Management System

A full-stack mini project demonstrating the **0/1 Knapsack algorithm** implemented in **C** with a
**Python Flask** backend and a polished **dark glassmorphism** web frontend.

---

## 📁 Project Structure

```
project/
│
├── app.py              ← Flask backend (routes, subprocess, JSON)
├── knapsack.c          ← 0/1 Knapsack in C (DP + traceback + DP table)
├── input.txt           ← Auto-generated by Flask before each run
├── output.txt          ← Written by the C program, read by Flask
│
├── templates/
│   └── index.html      ← Main HTML page (Jinja2 template)
│
├── static/
│   ├── style.css       ← Dark glassmorphism CSS
│   └── script.js       ← Frontend logic (add/delete/optimize/visualize)
│
└── README.md
```

## ⚙️ Setup & Installation

### Prerequisites
- Python 3.8+
- GCC (C compiler)
- pip

### 1. Install Python dependencies
```bash
pip install flask
```

### 2. Compile the C program manually (optional — Flask auto-compiles)
```bash
gcc -O2 -o knapsack knapsack.c
```

### 3. Run the Flask app
```bash
python app.py
```

### 4. Open in browser
```
http://127.0.0.1:5000
```

## 🖥️ How It Works

```
Browser (JS)  →  POST /optimize (JSON)
                        ↓
              Flask (app.py)
              • Writes input.txt
              • gcc compiles knapsack.c  (if needed)
              • subprocess runs ./knapsack
              • Reads output.txt
              • Returns JSON to browser
                        ↓
              C Program (knapsack.c)
              • Reads input.txt
              • Builds DP table (n+1 × W+1)
              • Traces optimal items
              • Writes output.txt
```

## 🧠 Algorithm — 0/1 Knapsack (Dynamic Programming)

**State:** `dp[i][w]` = maximum profit using first `i` items with capacity `w`

**Recurrence:**
```
dp[i][w] = dp[i-1][w]                                      if weight[i] > w
dp[i][w] = max(dp[i-1][w],  dp[i-1][w - weight[i]] + profit[i])   otherwise
```

**Traceback:** Walk back from `dp[n][W]` — if `dp[i][w] ≠ dp[i-1][w]`, item `i` was selected.

**Time Complexity:** O(n × W)  
**Space Complexity:** O(n × W)
